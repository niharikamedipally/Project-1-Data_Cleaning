# Titanic Data Cleaning & Visualization

## Project Overview
This project demonstrates the process of cleaning and visualizing the Titanic dataset using Python. The dataset was analyzed to identify missing values, duplicate records, and outliers before creating meaningful visualizations.

## Objectives
- Load the Titanic dataset
- Explore the dataset
- Handle missing values
- Remove duplicate records
- Handle outliers
- Visualize the data using charts

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Titanic Dataset from Kaggle

## Visualizations
- Survival Count
- Age Distribution
- Passenger Class Distribution
- Survival by Gender
- Correlation Heatmap

## Conclusion
The project demonstrates the complete process of data cleaning and visualization. It helps understand passenger information and provides meaningful insights using graphical representations.
